## Häufige Fragen

**Wie starte ich einen Online-Wettbewerb?**
Öffne die App → Online → Wettbewerb erstellen. 
Den angezeigten Code an die Schützen weitergeben.

**Wie treten Schützen bei?**
App öffnen → Online → Beitreten → 6-stelligen Code eingeben.

**Werden meine Daten gespeichert?**
Alle Daten werden lokal auf deinem Gerät gespeichert. 
Beim Online-Wettbewerb werden Ergebnisse temporär über iCloud übertragen.

**Wie lösche ich alle Daten?**
Einstellungen → Wettbewerb zurücksetzen.

**Die App stürzt ab / ein Fehler tritt auf**
Bitte beschreibe das Problem und deine iOS-Version in einer E-Mail an [deine@email.de]
